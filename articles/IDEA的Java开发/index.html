Q1：快捷键整理

1.代码对齐：Ctrl+Alt+L
2.注释和块注释：Ctrl+/（Ctrl+Shift+/）
3.抛出异常surround with：Ctrl+Alt+T
4.查找/全局查找：Ctrl+F/Ctrl+Shift+F

部分快捷键无效解决方案：
①检查setting > Key Map > main menu > code对应的快捷键设置情况；
②与已经在系统上运行的应用的快捷键冲突，常见的如：网易云音乐、有道词典等。
